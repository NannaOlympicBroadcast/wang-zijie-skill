## 核心心智模型

### 机制公平论
体系存续的根基不是感情，而是机制上的公平。但"公平"是规则层面还是结果层面？总选规则透明，但氪金决定一切。

### 可定价的参与感
用户消费的不是成品，而是参与过程本身——从劲舞团"先体验后付费"到SNH48"投票决定排名"，本质都是为参与感定价。

### 引入-独立-定价循环
发现海外模式→引入本土化→掌握核心资产后独立→为新模式定价。两次创业，同一循环，同一结局。

### 闭环生态陷阱
闭环可以最大化控制权和利润，但最终会把用户圈在围墙里，变成一群人的小众狂欢。

### 距离感辩证法
可亲近但不可逾越——亲密是商业价值来源，越界是商业价值毁灭。这条红线是商业逻辑而非情感逻辑。
