# 王子杰决策记录与商业行动

> 调研时间：2026年4月16日
> 重要说明：王子杰于2026年4月14日因突发心源性疾病去世，享年63岁。本文件记录其生前重大决策。

## 一、创立丝芭传媒与引入AKB48模式

### 决策背景
- 久游网游戏业务因腾讯《QQ炫舞》入局、市场监管收紧、用户审美变迁而渐入低谷
- 2010年，47岁的王子杰决定跨界偶像产业，创立丝芭传媒（前身久尚演艺）
- 当时国内偶像市场"三无"：无成熟偶像土壤、无粉丝消费习惯、无配套运营体系
- 外界普遍认为是"必输赌局"

### 决策逻辑
- 王子杰在日本留学及工作期间，与AKB48创始人秋元康有过私交，这是能拿到运营权的原因之一
- 坚信"可面对面的偶像"在中国有巨大市场
- **赌上全部资源**从上海星梦剧院起步
- 将游戏化运营思维深度植入偶像产业，用排名、养成、氪金、陪伴等逻辑重新定义造星

### 核心商业创新
- 开创"剧场公演+握手会+年度总选"三位一体运营体系
- **可面对面的偶像**——核心口号
- 粉丝通过消费决定偶像资源分配
- 单场年度总决选投票吸金超4亿元

### 结果
- SNH48成为中国首个大型女子偶像团体，成功破圈
- 丝芭传媒成为国内偶像经济头部企业

## 二、与AKS（AKB48运营方）分裂

### 决策过程与时间线

| 时间 | 事件 |
|------|------|
| 2012年 | 王子杰从日本AKB48本部拿到上海分团运营权，SNH48正式成立 |
| 2015年8月 | 丝芭注册北京丝芭文化传媒有限公司（即BEJ48经纪公司），并在武汉、沈阳、长沙、哈尔滨、广州等地布局8家公司，为独立开团做准备 |
| 2016年3月 | SNH48发布首张原创EP《源动力》，摆脱对AKB48曲库的依赖 |
| 2016年4月20日 | 丝芭单方面宣布在北京和广州成立BEJ48和GNZ48，引爆与AKS矛盾 |
| 2016年4月22日 | AKS宣布与酷酷音乐集团和中投中财基金战略合作，AKB48将全方位进军中国市场 |
| 2016年6月9日 | AKS发布声明，指责SNH48运营方违约，撤下官网所有SNH48相关内容 |
| 2016年6月10日 | SNH48回应强调"完全独立、自主经营"，双方仅是"跨国姐妹团体技术合作" |

### 决策逻辑（王子杰视角）

1. **从"代理运营"向"独立自主"的身份跃迁**：王子杰不认同AKS将丝芭视作"游戏代理公司"般的从属地位，强调丝芭拥有"完全独立、自主经营"的地位
2. **AKB48的"利用价值"已丧失**：
   - 热度引流作用降低——粉丝群体已从二次元宅男扩展到更广泛人群
   - 模式已烂熟于心——王子杰已完全掌握秋元康的运作模式
   - 曲库依赖被打破——原创EP《源动力》摆脱了对AKB48版权曲库的依赖
3. **控制权大于授权费**：拒绝将开发新团的权限交还给AKS，选择自己主导扩张
4. **风格转型**：SNH48形象风格从日系卡哇伊转向韩系女团的性感路线

### 历史重演：劲舞团模式的路径依赖
- **2007年（久游网时期）**：在代理韩国游戏《劲舞团》成功后，自主开发《超级舞者》《超级乐者》，试图绕过开发商单干，最终与T3和Yedang对簿公堂
- **2016年（丝芭时期）**：运营SNH48成功后，再次试图绕过日方通过原创歌曲和自主成立分团的方式"单干"
- **同一模式**：先代理获取经验 → 自建能力 → 摆脱对外部依赖 → 独立运营

### 结果
- SNH48彻底独立，不再是日本48系分团之一
- AKB48在上海另起炉灶做分团
- 粉丝分裂为三派：中立派、去日派、纯血派

## 三、创建BEJ48、GNZ48等姐妹团——"天下布妹"扩张策略

### 决策背景
- SNH48走红后，王子杰希望将偶像养成模式复制到全国
- 目标：覆盖更多城市、吸纳更多粉丝

### 决策逻辑
- 王子杰名言：**"打开电视全是我们的人"**
- 头部带腰部的出圈路线：头部成员通过演戏提升知名度，逐渐带动腰部成员破圈
- 本土化运营巩固受众粘性

### 执行
- 打造七大分团：SNH48、BEJ48（北京）、GNZ48（广州）、CKG48（重庆）、SHY48（沈阳）等
- 成员峰值超200人
- 被誉为"东方AKB"

### 结果与问题
- 短期扩大了品牌影响力
- 长期来看扩张过快，各地分团质量参差不齐
- 北京、重庆分团相继停运
- 只剩上海核心团队和广州GNZ48

## 五、战略转型：口袋48、直播、元宇宙、出海

### 5.1 口袋48 App（2015年）

| 维度 | 内容 |
|------|------|
| 决策动机 | 作为游戏人出身，对风向转变相当敏锐 |
| 做法 | 打造星粉交流渠道，汇集直播和聊天等功能；内部很多玩法与后来韩国"泡泡"相似 |
| 问题 | 推出太早，市场尚未形成统一共识，粉丝及偶像本人都充满抗拒 |
| 闭环效应 | 粉丝从外部渠道被引流后，最终聚集沉淀到该App中，使所有流量和收入被丝芭获取，但导致旗下艺人难以出圈 |

### 5.2 SNH48-IDFT直播队（2019年）

| 维度 | 内容 |
|------|------|
| 做法 | 成立"直播队"，成员没有公演站位，主要通过口袋48直播互动，靠粉丝投票投回有正式番号的队伍 |
| 评价 | 该玩法与当下团播类似，同样因时机过早未能奏效 |

**两次转型的共同问题**：一是推出过早，市场未成熟；二是闭环思维，缺乏外部导流。

### 5.3 元宇宙转型（2022年）

| 维度 | 内容 |
|------|------|
| 决策 | 控股公司更名为美踏控股，核心业务转型为面向WEB3.0和XR互联网时代的沉浸式互动社交元宇宙 |
| 筹备 | 2021年12月底已提前注册"48元宇宙""META48""四八元宇宙"等商标 |
| 五大方向 | 沉浸式互动体验、虚拟偶像养成平台、WEB3.0内容共创、虚拟资产变现、虚拟城市构建 |
| 核心逻辑 | 王子杰称"最娱乐的科技企业和最科技的娱乐企业" |
| 实际进展 | 推出虚拟偶像"花戎FAF Girls""Infinity Girls"；构建227平方公里的虚拟都市"花戎" |
| 效果 | 尚未见明显成效 |

### 5.4 电商与MCN（2021年）

- 2021年8月重组核心业务，跨界进军电商和新消费领域
- 成立电商女团"浪彩少女AfterWaves9"，推出国潮彩妆品牌"花戎FAF"
- 受阻：美妆行业投入高、同质化、获客成本高；上海疫情影响导致产品推迟上市

### 5.5 出海战略（2025年）

- 复合式发展策略：剧场模式+多形式提升估值+本土化运营+全球适配+AI技术赋能

## 六、合约模式与艺人管理——争议最大的决策

### 核心决策：超长合约+高额违约金

- 基础合约8年，甚至包含"20年影视独家代理"条款
- 被粉丝戏称"**内娱缅北**"
- 成员实际分成比例极低（约10%-20%），公司占大头

### 艺人解约潮

| 艺人 | 纠纷详情 |
|------|---------|
| **鞠婧祎** | 2024年6月约满解约，丝芭主张2018年《补充协议》将合约延至2033年；法院两次笔迹鉴定"无法确认签名真实性"；鞠婧祎2013-2021年收入超1.39亿元；税务举报经官方核查未发现问题 |
| **黄婷婷** | 解约后被判赔350万元违约金，因无力赔偿沦为失信人员 |

| 整体 | 十年间超20名艺人提起解约诉讼 |

## 七、公司衰败与股权冻结

### 财务恶化

| 指标 | 2024年 | 2025年前10个月 |
|------|--------|----------------|
| 营收 | 5.76亿元 | 3.54亿元（骤降） |
| 净利润 | 1984.3万元 | 净亏损3899.3万元 |

- 公司整体营收过于依赖鞠婧祎一人，离巢后无替代流量来源

- 2025年7月：上海市虹口区人民法院冻结丝芭传媒及关联主体近1.3亿元股权

### 运营萎缩
- 北京、重庆分团相继停运
- 剧场收入暴跌42%
- 曾数百人的团队，如今只剩2人在岗

## 八、决策特征总结

| 特征 | 表现 | 案例 |
|------|------|------|
| **豪赌式决策** | 两次"豪赌"式转型，每次都押上全部资源，且初期都赌赢了 | 劲舞团、SNH48 |
| **敏锐但过早** | 能预判趋势，但往往早于市场成熟期2-3年 | 口袋48（2015）、直播队（2019） |
| **闭环生态思维** | 习惯做封闭生态，缺乏外界接口导流，最终变成"一群人的小众狂欢" | 口袋48闭环、剧场闭环 |
| **路径依赖** | 同一决策模式反复使用：先代理获取经验→自建能力→摆脱外部依赖→独立运营 | 劲舞团→超级舞者；AKB48→SNH48独立 |
| **过度依赖单一资产** | 久游依赖劲舞团，丝芭依赖鞠婧祎，均未建立多元化营收 | 鞠婧祎贡献35%+营收 |
| **控制欲强** | 合约模式过于强硬（超长合约+高额违约金），资源分配不均 | "内娱缅北"标签、20年影视独家代理 |
| **战略清晰但执行断裂** | "头部带腰部"路线因鞠婧祎后再无出圈成员而断裂 | 鞠婧祎后再无顶流出圈 |
| **创新勇气** | 敢在空白领域试错，用商业实践推动产业成型 | 国内偶像产业开创者 |

### 核心矛盾
1. **创新者 vs 控制者**：开创新产业但用严苛合约约束从业者
2. **远见者 vs 时机者**：看到方向但往往太早出手
3. **独立者 vs 依赖者**：追求独立自主但过度依赖单一产品/人
4. **理想主义 vs 商业理性**：标榜"可面对面的偶像"但分成极低

## 九、与久游网决策的对比（历史重演模式）

| 维度 | 久游网（2003-2010） | 丝芭传媒（2010-2026） |
|------|-------------------|---------------------|
| **起始** | 代理韩国《劲舞团》 | 获日本AKB48运营权 |
| **爆发** | 同时在线破百万 | SNH48成国内头部女团 |
| **独立** | 自研《超级舞者》绕过开发商 | 自创BEJ48/GNZ48绕过AKS |
| **分裂** | 与T3/Yedang对簿公堂 | 与AKS公开决裂 |
| **衰落** | 腾讯入局+版权纠纷→上市折戟 | 鞠婧祎出走+解约潮→由盈转亏 |
| **依赖** | 营收过度依赖劲舞团 | 营收过度依赖鞠婧祎 |
| **结局** | 逐步收缩，王子杰卸任 | 估值缩水84%，股权被冻结 |

> **同一模式的两次重演：代理→独立→衰落，核心问题都是过度依赖单一资产且闭环思维导致出圈困难。**
